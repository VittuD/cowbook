# Cowbook

Cowbook is a packaged Python engine for multi-camera cow tracking and [ground-plane projection](calibration.md) in a fixed barn setup.

Cowbook stands on its own as a CLI and as a Python package. The package covers the [vision pipeline](pipeline.md) and the [execution flow](job-execution.md).

[Getting Started](getting-started.md) covers local runs, [Python Usage](python-usage.md) covers direct package use, [Package Boundaries](package-boundaries.md) defines the stable import surface, and [References and Attribution](references.md) records the upstream stack this project builds on.
